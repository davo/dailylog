- Git completion in command line: Save the file [.git-completion.bash](https://github.com/git/git/blob/master/contrib/completion/git-completion.bash) in ~/ and load in `.bash_profile`
 
  ```sh

    . ~/.git-completion.bash
  fi
  ```
